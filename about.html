About OpenReads

This application was built as a Phase 1 Capstone Project. It demonstrates the power of modern web tools by fetching real-time data from the Open Library API.

Key Features

Smart Search

Instantly find books by title or author using an asynchronous API integration.

Local Favorites

Persistent storage using LocalStorage so your favorite picks stay saved even after a refresh.

Mobile Ready

A mobile-first layout built with Tailwind CSS that looks great on any screen size.
