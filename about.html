About Book Explorer

This application was built as a Frontend Phase 1 Capstone. It utilizes the Open Library API to fetch real-time data and Tailwind CSS for a modern, responsive design.

Key Features

Searching Books

Search for books by title, author, or genre using our powerful search engine.

Add Books to Favorites

Save your favorite books to a personalized list for easy access later.

View and Remove Books

Remove books from your favorites list with a simple click.
